# Sokoban-Game-Demo
Sokoban Game Demo created using PhaserJs

Play the demo here: (https://eddornelas.github.io/Sokoban-Game-Demo/)

## How to play
User keyboard arrows or using mouse/touch swipe gestures (up, down, left, right)
